Introdução à Python

O que é Python ?

Python é uma popular linguagem de programação. Ela foi criada em 1991 por Guido van Rossum.

Ela é muito usada para:

  • Desenvolvimento web ;
  • Desenvolvimento de software ;
  • Matemática ;
  • Scripts de sistema .

O que pode ser feito com Python ?

  • Python pode ser usado num servidor para criar aplicações web ;
  • Python poder ser usado juntamente com softwares para criar fluxos de trabalho ;
  • Python pode ser conectado a sistemas de base de dados.Sendo também capaz de ler e modificar arquivos ;
  • Python pode ser usado para manipular big data e executar cálculos matemáticos complexos ;
  • Python pode ser usado para prototipagem rápida,ou para desenvolvimento de softwares "production-ready".

Estruturas de Decisão

Na programação, estruturas de decisão são utilizadas quando é necessária a escolha de um grupo de ações e estruturas que só devem ser executadas quando determinadas condições são ou não satisfeitas . Essa escolha determina o fluxo que o programa irá seguir .

Como exemplos de situações onde é necessário o uso de tais estruturas podem ser citadas ocasiões onde temos de verificar se um aluno passou por média ou se um número é primo .

Como Funciona ?

  • Analisa uma determinada condição ;
  • O resultado da análise sempre será : True(verdadeiro) ou False(falso) ;
  • Os operadores relacionais(+, -, = etc) são utilizados para montar uma condição ;
  • A condição pode ser simples ou composta ;
  • Os operadores and, or e not são utilizados .

Estruturas de Repetição

Na programação muitas vezes é necessária a iteração(repetição) de um ou mais comandos. Para isso podem ser usadas as chamadas estruturas de repetição.

Como exemplo, podem ser citadas situações onde devemos exibir uma determinada palavra várias vezes ou até mesmo fazer a leitura de 100 números. Sem o uso de repetições seriam necessários diversos inputs no mesmo código. Tornando algumas tarefas até mesmo inviáveis.

Como funciona ?

De forma geral, a repetição de comandos ocorre considerando :

  • Uma contagem pré-estabelecida de iterações, ou seja, quando se sabe o número de repetições ;
  • A ocorrência de uma determinada condição, sendo o número de repetições desconhecido ;

As linguagens de programação normalmente suportam os chamados laços de repetição(loops), que são estruturas que permitem executar repetidas vezes uma sequência de comandos .

Para criar tais laços em python, fazemos uso dos comandos for e while .

Sintaxe

Tradicionalmente, em diversas linguagens de programação o comando for permite executar instruções num número fixo de vezes .

Em python, o comando for permite percorrer itens de uma coleção e assim implementar a repetiçao para um número fixo de vezes .

for

 for variável in coleção: comandos else: comandos

range( )

A função range( ) é frequentemente utilizada com o comando for para controle das iterações. Ela retorna uma série numérica no intervalo enviado como argumento da função .

  • range(start, stop, step)
    • start - início da sequência
    • stop - último elemento da sequência(obrigatório)
    • step - intervalo entre os elementos

Exemplos

for

 #Sequência númerica de 0 a 3 for i in range(4): print(i)  
 0 1 2 3

Usando o step :

 #Sequência númerica de 0 a 3 com step 1 for i in range(0, 4, 1): print(i)  
 0 1 2 3

Step negativo :

 #Sequência númerica de 4 a 1 com step -1 for i in range(4, 0, -1): print(i)  
 4 3 2 1

Saltando de 2 em 2 números :

 #Sequência de 2 a 10 com step 2 for i in range(2, 11, 2): print(i) 
 2 4 6 8 10

while

O while executa uma sequência de comandos de acordo com uma condição pré-estabelecida .

 while condição: comandos else: comandos

Programa que solicite do usuário vários números até que seja informado o número 0. Sempre que o usuário informar um número o programa deverá mostrar o dobro desse número .

 n = int(input("Digite um número: ")) while(n != 0): print("O dobro de", n, "=", 2*n) n=int(input("Informe um número: ")) else: print("O usuário digitou zero") print("O programa terminou !")

Resultado :

 Digite um número: 2 O dobro de 2 == 4 Digite um número: 4 O dobro de 4 = 8 Digite um número: 0 O usuário digitou zero O programa terminou !

Vetores

Em diversas situações é necessário armazenar um grande volume de informações, o que torna impraticável a declaração de variáveis suficientes para armazenar esse grande volume .

  • Por exemplo :
    • Armazenar os nomes de 30 pessoas ;
    • Armazenar a descrição de 1000 produtos ;
    • Armazenar a idade de 1.000.000 de pessoas ;

Supondo que queremos ler 20 números inteiros, calcular e exibir a média dos números lidos. Se quiséssemos saber quais dos números lidos estavam acima da média, precisaríamos declarar 20 varíaveis ?

Felizmente com o uso de vetores podemos definir uma estrutura de dados para o armazenamento de vários valores, como no exemplo abaixo :

 vetor = ["a", "b", "c", "d", "e"]

No exemplo acima temos um vetor formado por 5 elementos, onde cada um é referenciado por um índice começando do 0. Ou seja "a" possui índice 0, "b" possui índice 1 e assim sucessivamente .

Os índices de um vetor nos permitem individualizar um elemento, o encontrando com mais facilidade .

Como funciona ?

Por definição vetor é um tipo especial de funcionalidade que permite armazenar vários valores ao mesmo tempo. Eles podem ser declarados através das função list( ), ou através do uso de [ ], ambos sendo guardados numa variável .

Declaração

 lista0 = [] #Lista vazia  lista1 = [1] #Lista com um elemento lista2 = [1,2] #Lista com dois elementos lista3 = list() lista4 = list(range(2, 11, 2))  # len - função para calcular a quantidade de # elementos contidos na lista  print("%s - %d" %(lista0, len(lista0))) print("%s - %d" %(lista1, len(lista1))) print("%s - %d" %(lista2, len(lista2))) print("%s - %d" %(lista3, len(lista3))) print("%s - %d" %(lista4, len(lista4)))

Resultado

 [] - 0 [1] - 1 [1, 2] - 2 [] - 0 [2, 4, 6, 8, 10] - 5

Manipulação de Vetores

[ ]

O [ ] pode ser usado para acessar os elementos da lista através do índice.

 lista = [10, 20, 30, 40] print(lista) print(lista[0]) print(lista[2:]) print(lista[:2]) print(lista[1:3])

Resultado

 [10, 20, 30, 40] 10 [30, 40] [10, 20] [20, 30]

insert

Adiciona elemento na lista, informando a posição e o elemento.

 lista = []  lista.insert(0, 10) lista.insert(0, 20) lista.insert(0, 30) lista.insert(0, 40)  print(lista)

Resultado

 [40, 30, 20, 10]

append

Adiciona um elemento no final da lista .

 lista = []  lista.append(10) lista.append(20) lista.append(30) lista.append(40)  print(lista)

Resultado

 [10, 20, 30, 40]

extend

Adiciona o(s) elemento(s) de uma lista no final de outra lista .

 lista1 = [10 ,20] lista2 = [30, 40]  print(lista1) print(lista2) lista1.extend(lista2) print(lista1) print(list2)

Resultado

 [10, 20] [30, 40] [10, 20, 30, 40] [30, 40]

+=

O += concatena listas .

 lista1 = [10, 20] lista2 = [30, 40]  print(lista1) print(lista2) lista1 += lista2 print(lista1)

Resultado

 [10, 20] [30, 40] [10, 20, 30, 40]

del

Remove determinado(s) elemento(s) da lista .

 lista = [10, 20, 30, 40] del(lista[2:]) print(lista)  lista = [10, 20, 30, 40] del(lista[:2]) print(lista)  lista = [10, 20, 30, 40] del(lista[1:3]) print(lista)  lista = [10, 20, 30, 40] del(lista[2]) print(lista)

Resultado

 [10, 20] [30, 40] [10, 40] [10, 20, 40]

remove

Remove determinado elemento da lista informando qual elemento deve ser eliminado .

 lista = [10, 10, 20, 20] print(lista) lista.remove(20) print(lista)

Resultado

 [10, 10, 20, 20] [10, 10, 20]

pop

Remove o último elemento da lista, aquele que possui o maior índice .

 lista = [10, 20, 30, 40] print(lista.pop()) print(lista.pop()) print(lista.pop()) print(lista.pop())

Resultado

 40 30 20 10

clear

 lista = [10, 20, 30, 40]  print(lista) lista.clear()  print(lista)

Resultado

 lista = [10, 20, 30, 40] []

count

Retorna a quantidade de determinado elemento na lista .

 lista = [10, 20, 30, 30, 40] print(lista) print(lista.count(10)) print(lista.count(30)) print(lista.count(50))

Resultado

 [10, 20, 30, 30, 40] 1 2 0

index

Verifica se um elemento está contido na lista. Retorna o index, se encontrado .

 lista = [10, 20, 30, 40] print(lista) print(lista.index(10)) print(lista.index(20)) print(lista.index(30)) print(lista.index(40))

Resultado

 [10, 20, 30, 40] 0 1 2 3

reverse

Inverte a ordem dos elementos na lista .

 lista = [10, 20, 30, 40]  print(lista) lista.reverse() print(lista)

Resultado

 [10, 20, 30, 40] [40, 30, 20, 10]

sort

Ordena os elementos da lista em ordem crescente .

 lista = [20, 40, 10, 30]  print(lista) lista.sort() print(lista)

Resultado

 lista = [20, 40, 10, 30] [10, 20, 30, 40]
